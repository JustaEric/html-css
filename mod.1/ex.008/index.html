Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag I (não semântica).

Nesta frase, temos um termo em ênfase usando a tag EM (semântico).

Texto marcado

Podemos também criar umtexto marcado usando a tag MARK

Mas se eu criar mais um texto marcado com a tag STYLE dentro do proprio MARK, não irá alterar a cor dos outros

Texto pequeno

Termo pequeno se usa a tag SMALL

Termo deletado

Podemos marcar um termo como excluído para marcar que ele deve ser lido, mas nao considerado.

Termo inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois.
